# Houston::Roadmaps

A module for Houston to facilitate managing milestones.


## Installation

In your `Gemfile`, add:

    gem "houston-roadmaps"

And in `config/main.rb`, add:

```ruby
use :roadmaps do
  # TODO: specify configuration options for Houston::Roadmaps here
end
```

And then execute:

    $ bundle
